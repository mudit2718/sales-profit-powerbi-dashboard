# 📊 Sales & Profit Analytics Dashboard

> An interactive Business Intelligence dashboard built using **Power BI**, **SQL Server**, and **Microsoft Excel** to analyze sales performance, customer segments, and product profitability.

---

## 📌 Project Overview

This project demonstrates an end-to-end Business Intelligence workflow by combining SQL Server, Excel, and Power BI to transform raw sales data into meaningful business insights.

The dashboard enables users to monitor KPIs, identify sales trends, evaluate customer segments, and analyze product performance through interactive visualizations.
