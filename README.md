# Sales & Profit Analytics Dashboard

## Project Overview
This project is an interactive Business Intelligence dashboard built using Power BI, SQL Server, and Excel.

## Tools Used
- Power BI
- SQL Server
- Microsoft Excel

## Dashboard Features
- KPI Cards
- Monthly Sales Analysis
- Sales Trend
- Customer Segmentation
- Product Performance
- Interactive Filters
